# Bee Online Marketing Website

A modern, responsive **digital marketing website** built using **React and Vite**.  
This project features multiple landing-page sections, including a hero banner, services overview, client impact, testimonials, technology partners, blog highlights, and a complete footer experience.

The website is designed with a **component-based architecture**, ensuring scalability, maintainability, and easy customization.

---

# Features

- Fully responsive design for **desktop, tablet, and mobile**
- **Modular React component architecture**
- Reusable UI components for faster development
- Centralized global styling for consistent design
- Optimized development workflow using **Vite**
- Clean and scalable project structure
- Smooth UI layout for marketing and business websites

---
